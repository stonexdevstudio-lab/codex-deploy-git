*   **Vite & React 18+ (TypeScript):** Modern, type-safe reactive architecture to guarantee crash-free browser rendering and blazing-fast interface switching.
*   **Tailwind CSS & Utility Frameworks:** Dynamic styling optimized for both high-end desktop resolutions and compact mobile/tablet touch targets.
*   **Lucide Icons & Micro-Interactions:** Custom modern iconography to represent domain settings, logistics metrics, and role configurations.

*   **Ingress & Routing Protection:** Custom Firebase security rules restricting operational reads/writes to verified roles.
